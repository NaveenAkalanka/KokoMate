## üöÄ Getting Started

```bash
# 1. Clone
git clone https://github.com/NaveenAkalanka/KokoMate.git
cd KokoMate

# 2. Install
npm install

# 3. Run dev server
npm run dev
```

### Build Android APK

```bash
# Build static export
npm run build

# Sync to Android
cd kokomate-android
npx cap sync android

# Open in Android Studio or build directly
cd android
./gradlew assembleRelease
```

